# json-server-posts-manager

A simple and interactive web application for managing posts using **JSON Server** as a mock backend.
This project demonstrates basic CRUD operations (Create, Read, Update, Delete) with a clean UI built using Bootstrap.

---

### 🚀 Features

- ➕ Add new posts
- ✏️ Edit existing posts
- ❌ Delete posts
- 📄 Display posts in a dynamic table
- 🔄 Real-time updates after each operation

### ⚙️ About JSON Server

JSON Server is a simple tool that allows you to create a full fake REST API with zero coding.
It is perfect for frontend developers who want to test their applications without building a real backend.

🔗 Official Website: https://github.com/typicode/json-server

---

### 📦 Installation & Run

1. Install JSON Server globally:

```bash
npm install ./srver/json-server
```

2. Run the server:

```bash
npx json-server ./server/db.json
```

3. Open your project in the browser.
